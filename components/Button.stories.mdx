# Button

Let's define a story for our `Button` component:

<Story name="positive" args={{ text: 'hello' }}>
<Button text="positive"/>
    </Story>

We can drop it in a `Canvas` to get a code snippet:

<Canvas>
  <Story name="negative">
    <Button text="negative"/>
  </Story>
</Canvas>

We can even preview multiple stories in a block. This
gets rendered as a group, but defines individual stories
with unique URLs and isolated snapshot tests.

<Canvas>
  <Story name="warning">
    {(args) => <Button text="warning" {...args} />}
  </Story>
  <Story name="neutral">
    <Button text="neutral"/>
  </Story>
  <Story name="error">
    <Button text="error"/>
  </Story>
  <Story name="with icon">
    {args => <Button {...args} />}
  </Story>
</Canvas>
